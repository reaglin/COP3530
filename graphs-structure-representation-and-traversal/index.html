COP3530 — Data Structures

Graphs: Structure, Representation, and Traversal

This module introduces graph data structures — one of the most versatile abstractions in computer science. You will explore how graphs are defined, how they can be stored efficiently using adjacency matrices and adjacency lists, and how to traverse them using Breadth-First Search (BFS) and Depth-First Search (DFS), culminating in a hands-on JavaScript implementation.
