COP3530 — Data Structures

Binary Trees: Structure, Traversal, and Operations

This module introduces the fundamental concepts of binary trees, exploring how nodes and edges form hierarchical structures, how binary search trees maintain ordering properties, and how classic traversal algorithms—in-order, pre-order, and post-order—navigate tree data. Learners will implement insertion, search, and traversal logic in JavaScript and develop an intuitive model for reasoning about recursive tree operations.
