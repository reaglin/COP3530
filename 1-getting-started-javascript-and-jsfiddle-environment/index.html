COP3530 — Data Structures

Module 1. Getting Started: JavaScript and JSFiddle Environment

This module lays the groundwork for the entire course. You will be introduced to JavaScript as the language of instruction, learn to navigate the JSFiddle browser-based coding environment, and write your very first program — all before any local tooling is required. Use the artifacts below as your guided roadmap through these foundational concepts.
