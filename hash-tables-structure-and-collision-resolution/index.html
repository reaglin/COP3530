COP3530 — Data Structures

Hash Tables: Structure and Collision Resolution

This module explores the design and internal workings of hash tables — one of the most widely used data structures in computer science. Topics span the fundamentals of key-value storage, hash function properties, and the core strategies used to handle collisions: chaining and open addressing.
