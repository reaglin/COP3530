COP3530 — Data Structures

Recursion: Concepts and Implementation

This module introduces recursion as a fundamental problem-solving paradigm in computer science. Learners will explore how recursive functions are structured, how the call stack manages execution, and how classic algorithms such as factorial and Fibonacci are implemented recursively — building an intuition for when and how to apply recursion effectively in data structures and beyond.
