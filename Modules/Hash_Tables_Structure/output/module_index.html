COP3530 — Data Structures

Hash Tables: Structure and Collision Resolution

This module introduces hash tables as a foundational data structure, examining their array-based design and key-value storage model. Learners will explore the properties of effective hash functions — including determinism, uniform distribution, and computational speed — and then work through the primary strategies for handling collisions: chaining and open addressing.
