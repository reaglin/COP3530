Lists and Arrays: Structure and Introduction to Big O Notation

This module establishes the foundations of two of the most essential data structures in computing — arrays and lists — and introduces the analytical tools needed to evaluate their efficiency. Learners will explore how arrays organise data in contiguous memory, how lists achieve dynamic flexibility, what the four core operations (access, search, insertion, deletion) look like in practice, and how Big O notation provides a principled language for comparing algorithmic performance as input size scales.
