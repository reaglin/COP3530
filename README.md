# COP3530 - Data Structures

This course introduces fundamentals of data structures. Topics include: programming strategies, data structures, searching, complexity, queues, sorting, and algorithms. Extensive programming is required.

## Modules

1. 1. Getting Started: JavaScript and JSFiddle Environment
2. 2 - Lists and Arrays: Structure and Introduction to Big O Notation
3. 3- Lists and Arrays: Algorithms and Operations
4. 4- The Stack: Structure and Use Cases
5. The Queue: Structure and Use Cases
6. Recursion: Concepts and Implementation
7. Computational Complexity: Deep Dive into Analysis
8. Sorting Algorithms: Comparison and Implementation
9. Hash Compression: Concepts and Techniques
10. Hash Tables: Structure and Collision Resolution
11. Hash Algorithms and Applications: Password Hashing and Beyond
12. Binary Trees: Structure, Traversal, and Operations
13. Advanced Tree Structures: AVL Trees, Heaps, and Tries
14. Graphs: Structure, Representation, and Traversal
15. Applied Data Structures: Blockchain, Digital Currency, and Beyond
