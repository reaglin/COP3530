COP3530 — Data Structures

Applied Data Structures: Blockchain, Digital Currency, and Beyond

This module bridges classical data structures theory with real-world applications in blockchain technology and digital currency systems. Learners will explore how foundational constructs — hash tables, trees, graphs, and linked lists — underpin modern decentralized architectures, from Merkle trees and cryptographic hashing to UTXO-based transaction models.
