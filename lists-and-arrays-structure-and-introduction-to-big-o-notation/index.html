COP3530 — Data Structures

Lists and Arrays: Structure and Introduction to Big O Notation

This module introduces the foundational concepts of arrays and lists as core data structures, covering memory layout, indexing, dynamic sizing, and essential operations. Learners will also be introduced to computational complexity and Big O notation as tools for reasoning about the efficiency of data structure operations.
